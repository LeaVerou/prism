Remove final line feed

Removes the final line feed in code blocks.

How to use

Obviously, this is supposed to work only for code blocks (<pre><code>) and not for inline code.

With this plugin included, any final line feed (and empty line) will be removed by default.

To bypass this behaviour, you may add the class keep-final-line-feed to your desired <pre>.
