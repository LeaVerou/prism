Linenums

Adds line numbers to highlighted code blocks.

Important! - The show-invisibles plugin is required for this plugin work properly.

How to use

Just add the attr data-linenums="1" to the pre element that wraps the code block.

<pre data-linenums="1">
	<code class="language-markup">
		[YOUR CODE]
	</code>
</pre>

Change the starting line number

Add the attr data-linenums-start="n" to the pre element that wraps the code block.

<pre data-linenums="1" data-linenums-start="98">
	<code class="language-markup">
		[YOUR CODE]
	</code>
</pre>
