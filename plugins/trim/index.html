Whitespace trimmer

Removes initial and tailing blank lines, end-of-line whitespace, and consistent indents.

How to use

Obviously, this is supposed to work only for code blocks (<pre><code>) and not for inline code.

With this plugin included, all periphery whitespace will be removed by default.

To bypass this behaviour for a particular <code> block, you may add one or more of the following classes to it or any ancestor:

dont-trim
Disables whitespace trimming entirely.
dont-trim-start
Preserve initial blank lines.
dont-trim-end
Preserve trailing blank lines.
dont-trim-trailing
Preserve whitespace at the end of each line.
dont-trim-indent
Preserve overall indentation.

Alternately, you can set global preferences in code by modifying Prism.plugins.trim.
